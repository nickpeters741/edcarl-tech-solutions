Installation of Network Structured Cabling

Network cables are using to connect network supported devices and transfer data and information between switched, routers, data storage, IP phones and other network products.

Edcarl Tech Solutions is a full-service network cabling provider with the resource to design and implement structured cabling in Kenya. Providing network cabling service throughout a business involve cable laying, installation of the connector – faceplates, Network cabinets and Testing of cables.Depend on the requirement cable will be the unshielded twisted pair (UTP), screened (foil) unshielded twisted pair (F/UTP), shielded twisted pair (STP), or multi mode or single mode fiber optics.

Network Structured Cabling types

Screened UTP Cabling

Indoor Ethernet applications commonly use Unshielded twisted pair (UTP) cables. Different twist rates are used to prevent interference or crosstalk

Shielded with FTP Cabling

The usage of copper cables is preferred in locations with high levels of disturbance as these prevent interference between various pairs and sets of cables.
